# 🧠 Mental Health Crisis Detector

A portfolio project using NLP and machine learning to detect mental health 
patterns in text. Built with Python, scikit-learn, and Streamlit.

## What it does
- Analyses text input and classifies it into 7 mental health categories
- Categories: Normal, Anxiety, Depression, Suicidal, Stress, Bipolar, Personality disorder
- Shows confidence scores for each prediction

## Tech stack
- Python, pandas, scikit-learn, Streamlit
- TF-IDF + Logistic Regression (76.12% accuracy)
- Dataset: 51,000+ mental health posts from Kaggle
